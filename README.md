## `Historia de PHP`   
**_¿Qué es PHP?_**
- [ ] Actualmente, **PHP** és un lenguaje de programación enfocado al desarollo web.   
**_¿Para que se creo PHP?_**  
- [ ] **PHP** se creó para el rastreo de las visitas al currículum online personal.
